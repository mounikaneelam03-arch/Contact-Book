Contact Book Application (CLI Based)
A simple and interactive Command Line Interface (CLI) Contact Book application built using Python.
This project allows users to add, view, search, and delete contacts efficiently using fundamental Python concepts.

Project Overview
The Contact Book application helps users manage their contact details directly from the terminal.
It is a menu-driven program that demonstrates the use of:
Lists
Dictionaries
Functions
Loops
Conditional Statements
This project is part of my Python learning journey and internship tasks.
Features
 -Add a New Contact
 -View All Contacts
 -Search Contact by Name
 -Delete a Contact
-Stores contacts in a list of dictionaries
Technologies Used
Python 3
CLI (Command Line Interface)
Core Python Concepts
